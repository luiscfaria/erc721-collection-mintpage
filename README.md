## Usage

### Env Variables

Create a .env file in then root and add the following

```
NEXT_PUBLIC_CONTRACT_ADDRESS=0xb4d81ec55D86AEc8814C9d662c3726C48f3C5bdA
NEXT_PUBLIC_MORALIS_API_KEY=<moralis api key>
```

### Install Dependencies

```
yarn install
```

### Run

```
# Run frontend
yarn dev
```
